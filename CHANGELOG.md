# Change Log

## [0.2.4] - 2022-08-03
### Changed

- 기존 `npx` 가 아닌 각 프로젝트에 설치된 `ts-fe-cli` 를 직접 수행하도록 변경하였습니다.

### Added

- `Add Feature Module` 기능이 추가되었습니다.
- 좌측 탐색기(explorer) 에서 typescript 나 폴더 계통에서 우클릭 하면 나타납니다.
